# Install

## Express generator

```shell
$ npm install express-generator -g
$ express --view=pug chatbox
$ cd chatbox
$ npm install
```

RUN

```shell
DEBUG=chatbox:* & npm start
```

然後在瀏覽器中載入 http://localhost:3000/。

## SOCKET.IO

```shell
$ npm install socket.io
```

## Run Project 

```shell
$ npm start
```
